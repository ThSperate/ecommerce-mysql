# Projeto E-commerce - Banco de Dados MySQL

## Descrição do Projeto
Este projeto é um banco de dados relacional desenvolvido em MySQL para um e-commerce. Ele gerencia informações de clientes, fornecedores, produtos e pedidos, de forma organizada e eficiente.

O projeto inclui:
- Tabelas para clientes, fornecedores, produtos, pedidos e itens de pedidos.
- Relacionamentos entre fornecedores e produtos, e entre pedidos e clientes.
- Scripts para criação do banco e inserção de dados de exemplo.
- Diagrama do banco de dados (opcional, para visualização).

---

## Estrutura do Projeto

- **schema/** → contém o arquivo `schema.sql` com a criação das tabelas.
- **sample/** → contém o arquivo `sample_data.sql` com dados de teste.
- **images/** → pasta para colocar o diagrama do banco (arquivo `diagrama.png`).

---

## Como usar

1. Abra o MySQL Workbench ou outro cliente MySQL.
2. Crie o banco de dados e execute o script `schema.sql`.
3. Execute o script `sample_data.sql` para inserir dados de teste.
4. Coloque o diagrama do banco na pasta `images/` (opcional, arquivo `diagrama.png`).
